# unibo-dtm-ml-2526-cervical-cancer-predictor
This ML project focuses on the prediction of indicators/diagnosis of cervical cancer. Based on a dataset composed of data from hospital patients, this project will carry out a multi-label binary classification task via logistic regression, random forest, XGBoost, SVM, multi-label k-NN. The model will predict the results of the 4 diagnostic tests.
